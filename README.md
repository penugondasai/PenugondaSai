📌 Project Description
This is a personal portfolio website designed for a Penetration Tester and VAPT Specialist. It presents professional skills, tools, projects, and experience in an elite hacker-dashboard style interface — complete with Matrix rain, a live 3D cyber attack globe, and terminal animations.
Built entirely in one single HTML file — no build tools, no frameworks, no dependencies except CDN libraries.

✨ Features

🔢 Animated Page Loader with percentage counter
🌧️ Matrix Rain Background (Japanese + hex characters)
🖱️ Custom Neon Cursor with hover morphing effect
⌨️ Typing Effect cycling through professional titles
💻 Terminal Simulation in the About section
📊 Animated Skill Bars triggered on scroll
🃏 3D Flip Project Cards with front/back reveal
🌍 Three.js Rotating Globe with live cyber attack arc animations
🧩 Interactive Particle Field that reacts to mouse movement

📱 Fully Responsive — mobile, tablet, and desktop
